# App Icons

โฟลเดอร์นี้สำหรับเก็บไฟล์ icon ของแอพ WMS Barcode Scanner

## การใช้งาน

1. **ใส่ไฟล์ icon**: เปลี่ยนชื่อไฟล์ PNG ของคุณเป็น `app_icon.png` และใส่ในโฟลเดอร์นี้
2. **ขนาดที่แนะนำ**: 256x256 pixels หรือใหญ่กว่า
3. **รูปแบบไฟล์**: PNG (รองรับความโปร่งใส)

## ตัวอย่างการตั้งค่า

```
assets/icons/
├── app_icon.png    # ไฟล์ icon หลักของแอพ
└── README.md       # ไฟล์นี้
```

## หมายเหตุ

- แอพจะโหลด icon อัตโนมัติเมื่อเริ่มต้น
- ถ้าไม่พบไฟล์ `app_icon.png` แอพจะใช้ icon เริ่มต้นของระบบ
- Icon จะแสดงทั้งในหน้าต่างแอพและ taskbar (Windows)
